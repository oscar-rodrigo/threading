# Threads Writing Tool

## Overview
**Threads Writing Tool** is a single-user web application designed to streamline the writing process. It collects notes from various sources (email, voice, manual entry), intelligently classifies them into thematic threads using Gemini LLMs, and synthesizes these threads into cohesive markdown documents.

## Tech Stack
*   **Frontend:** React, TypeScript, Vite, Tailwind CSS, Shadcn/ui
*   **Backend (Planned):** Supabase (PostgreSQL), Zapier (Ingestion), Edge Functions
*   **AI:** Gemini 2.0 Flash (Classification), Gemini 2.5 Pro (Synthesis)

## Getting Started
Navigate to the `frontend` directory to start the development server:
```bash
cd frontend
npm install
npm run dev
```
